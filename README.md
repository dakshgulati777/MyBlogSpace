A sleek, single-page blog web app built with **pure HTML, CSS, and JavaScript** — no frameworks, no dependencies, and fully functional offline with **localStorage**.
## ✨ Features

✅ **Minimal black & white UI**  
✅ **Parallax scrolling hero section**  
✅ **Lighting/glow effects** on welcome text and blog content  
✅ **Create, publish, and save blogs** directly in-browser  
✅ **LocalStorage-powered memory** (no backend needed)  
✅ **Responsive design** for desktop & mobile  
✅ **Sticky navigation menu** with smooth scrolling  
✅ **Hover/cursor effects** for interactive experience  

## 🧠 Technologies Used

* HTML5
* CSS3 (with animations, shadows, parallax)
* Vanilla JavaScript (DOM + localStorage)

---

## 💡 Use Cases

* Beginner-friendly project to learn web fundamentals
* Blogging tool for journaling or content prototyping
* Portfolio-ready UI/UX demo
* Creative experiment with parallax and lighting effects
